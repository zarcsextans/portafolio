# Portafolio de Actividades - Calculadoras y Animaciones Matemáticas

## Información del Proyecto

Este portafolio integra diversas actividades interactivas de HTML, CSS y JavaScript, diseñadas para demostrar conceptos matemáticos de forma dinámica y visual. Cada actividad está disponible directamente en la página principal mediante **iframes**, permitiendo al usuario interactuar con ellas sin necesidad de abrir múltiples archivos.

## Actividades Integradas

El portafolio incluye las siguientes actividades:

1. **Actividad #2:** Números complejos  
2. **Actividad #4:** Conversión de forma polar a forma exponencial  
3. **Actividad #6:** Expresiones algebraicas  
4. **Actividad #8:** Operaciones algebraicas  
5. **Actividad #10:** Productos notables  
6. **Actividad #12:** Factorización  
7. **Actividad #14:** Ecuaciones lineales  

Cada actividad está alojada en su propia carpeta (`actividad02`, `actividad04`, etc.) y se integra mediante iframes para mantener la coherencia visual y funcional del portafolio.
